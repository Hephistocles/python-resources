In IDLE, write the following:

import distutils.util
distutils.util.get_platform()

That will print "win32" if you're on 32 bit Python, or "win_amd64" if you're on 64 bit Python

Go to http://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame

if you're on 64 bit:
	Click "pygame-1.9.2a0-cp34-none-win_amd64.whl"
else:
	Click "pygame-1.9.2a0-cp34-none-win32.whl"

Save the file in C:\Python34\Scripts (make sure not to change the name!)

Start > cmd

type: cd "C:\Python34\Scripts"
if 64 bit:
	type: pip3 install pygame-1.9.2a0-cp34-none-win_amd64.whl
else:
	type: pip3 install pygame-1.9.2a0-cp34-none-win32.whl
